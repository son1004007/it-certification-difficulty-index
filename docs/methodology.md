# 난이도 산정 기준

## 기본 전제

- IT 기본지식이 있는 응시자를 기준으로 합니다.
- 순공부시간에는 강의, 교재, 공식문서, 기출, 실습, 모의고사를 포함합니다.
- 경력요건 충족에 필요한 실제 근무기간은 학습시간에 포함하지 않습니다.
- 자격증 1개는 H1~H6 중 하나의 난이도만 가집니다.
- 난이도는 공식 등급이 아니라 학습 로드맵과 역량 비교를 위한 추정값입니다.

## 난이도 기준

| 등급 | 점수 | 대표 학습시간 | 설명 |
|---|---:|---:|---|
| H1 | 1 | 40h | 입문, 리터러시, 기본 개념 이해 중심 |
| H2 | 2 | 80h | 기초 개념과 기본 실습 또는 기출 반복 필요 |
| H3 | 3 | 150h | 실무 기초, 시나리오 이해, 문제풀이 또는 실습 필요 |
| H4 | 4 | 275h | 넓은 범위, 실기형 또는 실무 판단형 문제 포함 |
| H5 | 5 | 500h | 전문 실무경험, 고급 설계, 감사, 튜닝, 운영 판단 필요 |
| H6 | 6 | 1,000h+ | 논술, 구술, 아키텍처, 경영 판단, 장기 실무경험 요구 |

## 보정 요소

최종 난이도는 단순 학습시간 외에 다음 요소를 반영합니다.

- 실기형 시험 여부
- 논술형 또는 구술형 여부
- 시나리오 판단 문제 비중
- 경력요건 존재 여부
- 시험범위의 폭
- 특정 벤더나 플랫폼 실습 필요성
- 합격률이 공개된 경우의 참고값

## 판정 방식

```text
자격증별 대표 학습시간 산정
→ 시험유형, 실기, 논술, 경력요건, 플랫폼 의존성 보정
→ H1~H6 중 하나의 최종 난이도 부여
```

## 예외 처리

- 은퇴 또는 대체 예정 자격증은 `status: retired` 또는 `status: replaced`로 표시합니다.
- 신규 시험은 공식 출제범위와 유사 자격증을 기준으로 임시 분류합니다.
- 시험 개편이 확인되면 `last_verified` 또는 기준일을 갱신합니다.

## 해석 주의사항

같은 난이도라도 시험 성격은 다를 수 있습니다.

| 예시 | 같은 점 | 다른 점 |
|---|---|---|
| CKA / CISA | 둘 다 H4 | CKA는 Kubernetes 실기형, CISA는 감사·통제 판단형 |
| SQLP / CISSP | 둘 다 H5 | SQLP는 DB 튜닝, CISSP는 보안관리·아키텍처 판단 |
| PMP / 정보보안기사 | 둘 다 H4 | PMP는 프로젝트관리 시나리오, 정보보안기사는 보안기술·법규 |
