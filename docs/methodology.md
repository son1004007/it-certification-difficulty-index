# 난이도 산정 기준

## 기본 전제

- 이 저장소의 난이도는 **자격증 취득 난이도**만 평가합니다.
- 기준 대상은 IT 기본지식이 있는 응시자입니다.
- 순공부시간에는 강의, 교재, 공식문서, 기출, 실습, 모의고사를 포함합니다.
- 경력요건 충족에 필요한 실제 근무기간은 학습시간에 포함하지 않습니다.
- 자격증 1개는 H1~H10 중 하나의 난이도만 가집니다.
- 난이도는 공식 등급이 아니라 취득 준비 부담을 비교하기 위한 추정값입니다.

## 제외 기준

아래 요소는 난이도 산정에서 제외합니다.

- 취득 이후 실제 업무 활용 수준
- 시장 가치
- 연봉 영향
- 직무 전환 효과
- 프로젝트 투입 가능성
- 자격 취득 후 숙련도

## 난이도 기준

| 등급 | 점수 | 대표 학습시간 | 설명 |
|---|---:|---:|---|
| H1 | 1 | 40h | 입문 개념과 기본 용어 중심 |
| H2 | 2 | 80h | 기초 개념과 기출 반복으로 취득 가능한 수준 |
| H3 | 3 | 130h | 시험범위가 넓어 기본서와 문제풀이가 필요한 수준 |
| H4 | 4 | 180h | 필기와 실기 또는 시나리오형 문제 대비가 필요한 수준 |
| H5 | 5 | 260h | 범위, 문제해석, 작업형 또는 핸즈온 부담이 큰 수준 |
| H6 | 6 | 400h | 전문 영역의 고급 문제와 장기 대비가 필요한 수준 |
| H7 | 7 | 550h | 낮은 합격률, 경력요건, 복합 시나리오가 반영되는 수준 |
| H8 | 8 | 700h | 장기 학습과 고급 문제해석이 필요한 전문가 수준 |
| H9 | 9 | 1,000h | 논술, 구술 또는 장기 누적 학습이 필요한 최상위 수준 |
| H10 | 10 | 1,200h+ | 기술사급 장기 준비가 필요한 최고난도 수준 |

## 반영 요소

최종 난이도는 다음 요소를 반영합니다.

- 합격권 도달에 필요한 순공부시간
- 공개 합격률 또는 알려진 합격 난이도
- 시험범위의 폭
- 필기, 실기, 작업형, 핸즈온 여부
- 논술형 또는 구술형 여부
- 시나리오형 문제 비중
- 응시 또는 자격인증 경력요건 존재 여부
- 시험 개편, 은퇴, 대체 여부

## 판정 방식

```text
합격권 도달 학습시간 산정
→ 합격률, 시험형태, 경력요건, 문제유형 보정
→ H1~H10 중 하나의 취득 난이도 부여
```

## 예외 처리

- 은퇴 또는 대체 예정 자격증은 `status: retired` 또는 `status: replaced`로 표시합니다.
- 신규 시험은 공식 출제범위와 유사 자격증을 기준으로 임시 분류합니다.
- 시험 개편이 확인되면 기준일 또는 데이터 파일을 갱신합니다.

## 해석 주의사항

같은 난이도라도 시험 성격은 다를 수 있습니다.

| 예시 | 같은 점 | 다른 점 |
|---|---|---|
| CKA / PMP | 둘 다 H5 | CKA는 핸즈온 시험, PMP는 프로젝트관리 시나리오형 시험 |
| SQLP / CISSP | 둘 다 H7 | SQLP는 SQL 튜닝 중심, CISSP는 보안 도메인 전반 중심 |
| 빅데이터분석기사 / AWS Data Engineer Associate | 둘 다 H4 | 빅데이터분석기사는 필기와 작업형, AWS Data Engineer Associate는 AWS 데이터 서비스 중심 |
