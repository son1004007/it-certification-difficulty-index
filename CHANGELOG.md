# Changelog

## 2026-05-24

### Changed

- 난이도 체계를 H1~H10으로 세분화했습니다.
- 난이도 기준을 자격증 취득 난이도로 정리했습니다.
- 문서에서 실무 기준 표현을 제거했습니다.
- `data/certifications.yml`의 자격증별 난이도 값을 H1~H10 기준으로 재배치했습니다.
- README와 docs 문서를 H1~H10 기준으로 갱신했습니다.
